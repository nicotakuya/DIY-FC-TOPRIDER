# DIY-FC-TOPRIDER
![Image](https://github.com/user-attachments/assets/8a750178-72dc-4a55-94db-0df73971cbb7)

## OVERVIEW
ファミコン用ソフト「トップライダー」を遊ぶための専用コントローラの互換品です。Arduino(ATmega328/5V/16MHz)を使って自作しました。

## 操作方法
・アナログスティック（左右方向でバイクを傾ける。下方向に入れるとウィリー）

・アクセルボタン（最高の加速に設定しています）

・ブレーキボタン

・シフトスイッチ（ギヤのLow/Highを切り替えるスライドスイッチ）

・セレクトボタン

・スタートボタン
